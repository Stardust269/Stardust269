# 项目说明：三月样本（20260715）

> 同事参考：`sql/yye_pril_bal_sample_reference_20260715.sql`  
> 我方实现：`sql/run_all_20260715.sql`

---

## 一、样本入口

**左表**：`yye_pril_bal_info_20260715_1`（同事 reference：rk=1 + no_balance）

我方在 `_1` 上算 had/with → `jcr_pril_bal_info_20260715` → cohort 五条件。

---

## 二、处理链路

| 步骤 | 产出表 | 作用 |
|------|--------|------|
| 前置 | `yye_pril_bal_info_20260715_1` | 同事 _0/_1 |
| info | `jcr_pril_bal_info_20260715` | had/with，锚点 **days_dt** |
| cohort | `jcr_cohort_20260715` | info 五条件一层 `WHERE` |
| 特征+标签 | `jcr_credit_feature_label_full_20260715` | 终表 |

---

## 三、锚点

| 用途 | 锚点 |
|------|------|
| `no_balance_flg_*` / `had_*` / `with_*` | `days_dt` |
| 征信特征 / `zx_balance_label` | `days_dt` |

---

## 四、cohort 五条件

```sql
where crdt_lim_yx >= 20000
  and had_0_30_zx = 1 and had_31_60_zx = 1
  and no_balance_flg_60 = 1
  and with_0_30 + with_31_60 = 0
```

---

## 五、数据集划分

| 集合 | 条件 |
|------|------|
| train | 全 cohort，`hash(uuid+dt) % 10 < 8` |
| val | 全 cohort，其余 20% |

**无离线 test 集**；模型效果后续直接做产品实验验证。

---

## 六、核验

```sql
select m, count(1), count(distinct uuid)
from jcr_cohort_20260715 group by m order by m;

select m, dataset_split, count(1)
from jcr_credit_feature_label_full_20260715
group by m, dataset_split order by m, dataset_split;
```
